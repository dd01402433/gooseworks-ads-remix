# Gooseworks Ads Remix 廣告概念生成技能

## 技能名稱
`gooseworks-ads-remix`

## 觸發條件
當用戶提供品牌/項目資訊並要求產出廣告概念時觸發。典型關鍵詞包括：
- "廣告概念"、"廣告文案"、"廣告素材"
- "Meta Ads"、"Google Ads"、"Facebook 廣告"
- "Hook"、"廣告發想"、"創意發想"
- "競品分析"、"廣告研究"
- "Gooseworks"、"Ads Remix"

---

## 執行流程

### Phase 1：資訊收集與品牌分析

**1.1 品牌資訊提取**
從用戶提供的資訊中提取：
- 品牌名稱、產品/服務類別
- 目標受眾（年齡、性別、興趣、痛點）
- 品牌調性關鍵詞（如：專業 / 親民 / 潮流 / 奢華）
- 主要價值主張（USP）
- 現有廣告素材或過往成功案例（如有）

**1.2 網站內容分析**（若用戶提供網站 URL）
- 使用 `web_fetch` 抓取網站首頁及關鍵頁面
- 分析：品牌語氣、視覺風格、核心訊息、CTA 模式
- 提取：標語、產品描述、客戶評價、信任信號

**1.3 品牌調性歸納**
將收集到的資訊整理為品牌調性卡片：
```
品牌調性：
- 語氣：________
- 核心價值：________
- 受眾痛點：________
- 差異化優勢：________
```

### Phase 2：競品研究（Competitor Research）

**2.1 競品識別**
- 基於產品類別與目標市場，使用 `web_search` 搜尋直接競品與間接競品
- 搜尋格式：`[產品類別] [目標市場] 競爭品牌 廣告案例`

**2.2 競品廣告分析**
對每個主要競品（3-5 個），分析：
- 廣告平台分佈（Meta / Google / TikTok）
- 常用 Hook 類型（問題型 / 好奇心型 / 痛點型 / 社會證明型）
- 視覺風格與文案策略
- CTA 模式與漏斗設計
- 優勢與可借鑑之處

**輸出**：競品廣告策略對照表

| 競品 | Hook 類型 | 核心訊息 | 視覺風格 | CTA | 優勢 | 弱點 |
|------|-----------|----------|----------|-----|------|------|

### Phase 3：廣告概念生成（結合 Gooseworks Ads Remix 方法論）

**3.1 概念框架**
基於 Gooseworks Ads Remix 的核心方法論，每個廣告概念包含：
- **Hook（0-3 秒）**：抓住注意力的開場
- **Body（3-25 秒）**：傳遞價值、展示產品
- **CTA（25-30 秒）**：明確行動呼籲
- **格式**：9:16 直式短影音（30 秒）/ 1:1 或 4:5 靜態素材

**3.2 方向生成**
針對每個廣告概念方向，必須包含：

**(A) Hook 庫（每個方向 3-5 個 Hook 變體）**
- 問題型 Hook：「你是否也曾_______？」
- 好奇心 Hook：「99% 的人不知道_______」
- 痛點型 Hook：「受夠了_______？」
- 社會證明 Hook：「超過 10,000 人已經_______」
- 對比型 Hook：「Before vs After」
- 數據型 Hook：「根據最新研究，_______」

**(B) 廣告文案（3 個完整版本）**
- 版本 A：情感驅動型
- 版本 B：數據/理性驅動型
- 版本 C：社會證明/Urgency 型

**(C) 平台適配**

| 平台 | 格式 | 最適 Hook | 文案長度 | 注意事項 |
|------|------|-----------|----------|----------|
| Meta Ads (FB/IG) | 9:16 / 1:1 / 4:5 | 好奇心 / 痛點型 | Primary Text 125 字內 | 頭 3 行是關鍵 |
| Google Ads | 文字 + 圖片 | 數據 / 問題型 | 標題 30 字 / 描述 90 字 | 關鍵字意圖匹配 |
| TikTok Ads | 9:16 直式 | 原生感 / 趨勢型 | 簡短口語化 | 前 2 秒決定留存 |
| YouTube Ads | 16:9 / 9:16 | 故事型 / 教學型 | 15-30 秒 | 可跳過前 5 秒 |

**(D) SEO 關鍵字策略**
- 主要關鍵字（高意圖）：購買意圖強的詞
- 長尾關鍵字：問題型 / 比較型搜尋詞
- 否定關鍵字建議（Google Ads）
- 廣告文案中可自然融入的關鍵字

**(E) 視覺方向建議**
- 色調 / 風格參考
- 關鍵畫面描述
- Gooseworks 素材生成時的風格提示詞

## 使用限制與注意事項

1. **品牌資訊必須由用戶提供**：網站 URL、品牌調性、目標受眾等核心資訊不可憑空編造
2. **競品研究基於公開資訊**：僅分析可公開獲取的競品廣告素材與網站資訊
3. **Gooseworks 素材生成需外部平台執行**：本技能產出的是概念策略文檔，實際影片素材需透過 Gooseworks 平台生成
4. **合規性**：廣告文案需符合各平台廣告政策（Meta / Google 廣告政策）
5. **文化適配**：若目標市場為特定地區，需考量當地文化用語與禁忌
