# Gooseworks Ads Remix

廣告概念生成技能 — 結合 Gooseworks Ads Remix 能力，四階段執行流程：

**Phase 1** — 品牌網站分析 + 調性分析  
**Phase 2** — 競品研究對照表  
**Phase 3** — 廣告概念生成（Hook 庫 + 多方向文案 + 平台適配 + SEO）  
**Phase 4** — 結構化輸出整合（PDF 報告）

每個廣告方向產出 3 個版本（情感 / 理性 / Urgency），3-5 個 Hook 變體，覆蓋 Meta / Google / TikTok / YouTube 四平台。
